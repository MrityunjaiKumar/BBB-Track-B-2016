#Pin mapping  
#####Seven segment display pinouts
LED           | BBB P9 pin
:-----------: |:-----------:
a             | "P9_15"
b             | "P9_16"
c             | "P9_21"
d             | "P9_23"
e             | "P9_27"
f             | "P9_13"
g             | "P9_11"  
#####Other peripherals
Component       | Corresponding pin
:--------------:|:-----------------:
LED             |"P9_14"
Push Button     |"P9_41"
LDR             |"AIN6", "P9_35"
Thermistor (M02)|"AIN4", "P9_33"
Potentiometer   |"AIN3", "P9_38"
